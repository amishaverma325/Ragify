# IT Support Guide

## Password Reset
Use the Self-Service Portal first.
If portal access fails, contact IT on Slack channel #it-help with your employee ID.

## Laptop Replacement
Standard laptop refresh cycle is every 3 years.
Immediate replacement is allowed for hardware failure confirmed by IT diagnostics.

## Software Requests
Approved software list is maintained by IT procurement.
New software requests need business justification and manager approval.
Typical approval time is 2-5 business days.
